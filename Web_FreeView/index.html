实验流程

注视点：屏幕中央先出现空心圆圈，按空格键后变为实心黑点，短暂等待后开始实验。

目标识别：注视点变为实心后，迅速在圆形背景花纹中搜寻小光斑：
一旦发现光斑，请尽快按空格键，并持续盯着它直到背景消失。
按键后：背景会闪烁两次，然后再出现一个闪烁的圆圈提示真实目标位置。

  • 5秒内未找到，试次会自动结束并进入下一个。未找到请不要按键！
  • 实验结束后会自动下载 JSON 文件；若需要 CSV / TXT，可在结束页面手动下载。
